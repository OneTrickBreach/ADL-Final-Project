# ADL-Final-Project
ADL CS7180 Final Project

Group Members: Elizabeth Coquillette, Nishant Suresh, Ishan Biswas
Initial Project Idea: Perception on Images using RL

## Project Overview
Multi-agent RL for Keep Away Zone (KAZ) using Ray RLlib.

## Structure
- `data/`: Data storage
    - `raw/`: Original, immutable data
    - `processed/`: Cleaned and transformed data
- `src/`: Source code and scripts
- `notebooks/`: Exploratory analysis and prototyping
- `models/`: Trained models and checkpoints
- `docs/`: Documentation
- `results/`: Output files (plots, metrics, logs)
- `.venv/`: Virtual environment
